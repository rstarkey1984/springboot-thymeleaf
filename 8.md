# 8교시 - 표현식 연습 전용 페이지 매핑

## 학습 목표
- 기존 로그인/회원가입 흐름과 분리된 연습 전용 페이지를 만들 수 있다.
- 더미데이터 기반으로 Thymeleaf 표현식을 안전하게 연습할 수 있다.

## 학습 범위
- 연습 전용 URL 매핑
- 연습 전용 템플릿 작성
- 더미데이터를 이용한 표현식 확인

## 핵심 개념
- 운영 흐름(로그인/회원가입)과 연습 흐름을 분리하면 실습이 안정적이다.
- 표현식 연습은 전용 페이지에서 반복해도 본 기능에 영향이 없다.

## 실습 절차
1. `/practice/thymeleaf` GET 매핑을 만든다.
2. 컨트롤러에서 더미데이터를 `Model`에 담는다.
3. `practice-thymeleaf.html` 템플릿을 만든다.
4. 브라우저에서 연습 페이지를 열어 표현식을 확인한다.

## 실습 코드

### controller/PracticeController.java
```java
package com.example.lms.controller;

import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;

import java.util.List;

@Controller
public class PracticeController {

    @GetMapping("/practice/thymeleaf")
    public String practice(Model model) {
        model.addAttribute("pageTitle", "Thymeleaf 연습 페이지");
        model.addAttribute("username", "student01");
        model.addAttribute("successMessage", "처리 성공 메시지 예시");
        model.addAttribute("errorMessage", "처리 실패 메시지 예시");
        model.addAttribute("errorList", List.of("아이디는 필수입니다.", "비밀번호는 4자 이상이어야 합니다."));
        return "practice-thymeleaf";
    }
}
```

### templates/practice-thymeleaf.html
```html
<!doctype html>
<html lang="ko" xmlns:th="http://www.thymeleaf.org">
<head>
  <meta charset="UTF-8">
  <title>표현식 연습</title>
</head>
<body>
<h1 th:text="${pageTitle}">연습 페이지</h1>

<p th:if="${successMessage}" th:text="${successMessage}"></p>
<p th:if="${errorMessage}" th:text="${errorMessage}"></p>

<input type="text" name="username" th:value="${username}">

<ul>
  <li th:each="msg : ${errorList}" th:text="${msg}"></li>
</ul>

<a th:href="@{/login}">로그인 페이지로 이동</a>
</body>
</html>
```
