# 7교시 - 로그인/회원가입 화면 개선과 사용자 피드백

## 학습 목표
- 로그인/회원가입 화면의 사용성을 개선할 수 있다.
- 성공/실패/로그아웃 흐름에 맞는 안내 문구를 설계할 수 있다.

## 학습 범위
- 로그인 성공/실패 문구 설계
- 회원가입 완료 안내 문구 설계
- 로그아웃 안내 문구 설계
- 화면 간 이동 링크 정리

## 핵심 개념
- 사용자 피드백은 기능 완성도에 직접적인 영향을 준다.
- 컨트롤러에서 전달하는 메시지 키를 설계하고 바로 템플릿(`th:if`, `th:text`)에 반영한다.
- 자주 사용하는 표현식은 기능 구현 단계에서 즉시 적용한다.

## 실습 절차
1. 컨트롤러에서 사용할 메시지 키를 정의한다.
2. 로그인/회원가입/로그아웃 처리 시 메시지를 설정한다.
3. 로그인/회원가입 화면 문구와 링크 구조를 정리한다.
4. 실제 동작 흐름을 점검한다.

## 실습 코드

### controller/AuthController.java (일부)
```java
@PostMapping("/signup")
public String signupSubmit(
        @RequestParam String username,
        @RequestParam String password,
        @RequestParam String name,
        RedirectAttributes redirectAttributes
) {
    authService.signup(username, password, name);
    redirectAttributes.addFlashAttribute("signupSuccessMessage", "회원가입이 완료되었습니다. 로그인해 주세요.");
    return "redirect:/login";
}

@PostMapping("/login")
public String loginSubmit(
        @RequestParam String username,
        @RequestParam String password,
        RedirectAttributes redirectAttributes
) {
    if (authService.login(username, password)) {
        redirectAttributes.addFlashAttribute("successMessage", "로그인 성공");
        return "redirect:/login";
    }
    redirectAttributes.addFlashAttribute("errorMessage", "아이디 또는 비밀번호가 올바르지 않습니다.");
    return "redirect:/login";
}

@GetMapping("/logout")
public String logout(HttpSession session, RedirectAttributes redirectAttributes) {
    session.invalidate();
    redirectAttributes.addFlashAttribute("logoutMessage", "로그아웃되었습니다.");
    return "redirect:/login";
}
```

### templates/login.html
```html
<!doctype html>
<html lang="ko" xmlns:th="http://www.thymeleaf.org">
<head>
  <meta charset="UTF-8">
  <title>로그인</title>
</head>
<body>
<h1>로그인</h1>
<p th:if="${signupSuccessMessage}">회원가입이 완료되었습니다. 로그인해 주세요.</p>
<p th:if="${successMessage}">로그인 성공</p>
<p th:if="${errorMessage}">아이디 또는 비밀번호가 올바르지 않습니다.</p>
<p th:if="${logoutMessage}">로그아웃되었습니다.</p>

<form method="post" action="/login">
  <label>아이디 <input type="text" name="username"></label><br>
  <label>비밀번호 <input type="password" name="password"></label><br>
  <button type="submit">로그인</button>
</form>

<a href="/signup">회원가입으로 이동</a>
</body>
</html>
```
